Clean Server 指南

规则说明

本服务器为 无规则服务器 (Anarchy)。

  • 允许自由 PVP / 抄家
  • 允许建立任意红石机器
  • 没有领地保护，请隐藏好你的基地！

关于登录插件

  • /login 密码 或 /l 密码 - 登录
  • /register 密码 重复密码 或 /reg 密码 重复密码 - 注册
  • /changepassword 旧密码 新密码 重复新密码 或 /changepw 旧密码 新密码 重复新密码 - 更改密码

关于皮肤插件

  • 正版账户插件可直接从mojang下载皮肤并正常显示
  • 离线账户不能显示客户端本地设置的皮肤
  • 所有玩家均可使用/skins打开皮肤菜单 选择/更换皮肤 查看历史皮肤
  • 提供多种热门皮肤可供选择!

常用指令(tpa指令)

  • /tpa <玩家名> - 请求传送
  • /tpahere <玩家名> - 请求传送到自己的位置

关于派系(工会)插件

  • 在游戏内使用/team ?查看指南
  • 任何玩家有权利加入/建立/退出队伍
  • 所有玩家可使用/team guiGUI菜单
  • 该插件只作为促进玩家合作的功能补充 不影响服务器无规则内核
  • 补充指南:
  • /team sethome或setwarp设置家/传送点
  • /teamchat 或 /tchat 或 /tc对内聊天

常用指令(tpa指令)

  • /tpa <玩家名> - 请求传送
  • /tpahere <玩家名> - 请求传送到自己的位置
